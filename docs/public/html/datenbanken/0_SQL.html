SQL Projekt

SQL übt man am besten mit einer echten Datenbank und nicht auf dem Papier. Dafür habe ich eine Docker-Umgebung erstellt, die alles Notwendige enthält.

  • Das Projekt: Docker-SQL

Voraussetzungen

Das Projekt lässt sich mit einem Knopfdruck aufbauen, wenn folgende Bedingungen erfüllt sind:

  • Docker ist installiert.

  • VSCode ist installiert.

    • Das Plugin "Dev Containers" aus den VSCode Extensions ist installiert.

Projekt starten

Wenn alle genannten Voraussetzungen erfüllt sind, wird beim Öffnen des Ordners in VSCode ein Vorschlag erscheinen, den Container zu bauen. In der README-Datei steht, wie man den Container nutzt.

  • Man kann den Container auch manuell bauen, indem man auf das blaue Icon links unten in VSCode klickt () und "Reopen in Container" auswählt.
